# ❌ Situación Actual

## Problemas que enfrentamos hoy:

- **Gestión manual** de correspondencia física y electrónica
- **Sin trazabilidad** de documentos entre oficinas
- **Pérdida de documentos** y falta de control
- **Sin sistema de archivo** organizado por TRD
- **Dificultad para cumplir** tiempos de respuesta legales
- **Duplicación de esfuerzos** entre áreas
- **Búsqueda de documentos** consume tiempo valioso

---

# ✅ La Solución

## Sistema Integrado de Gestión Documental

**Dos módulos principales:**

1. 🏥 **Gestión Documental** - Archivo y organización
2. 📧 **Correspondencia** - Radicación y seguimiento

**100% Web | Acceso desde cualquier computador | Sin instalación**

---

# 🏥 Módulo 1: Gestión Documental

## Funcionalidades Implementadas

✅ **Registro de Archivos**
- Clasificación por Series y Subseries (TRD)
- Gestión de documentos físicos y electrónicos
- Control de ubicación (cajas, carpetas, tomos)

✅ **FUID** (Formato Único de Inventario Documental)
- Inventarios documentales organizados
- Historias clínicas clasificadas
- Exportación a Excel

✅ **Control de Acceso**
- Permisos por oficina
- Cada usuario ve solo lo de su área

---

# 📧 Módulo 2: Correspondencia

## 1. Radicación Automática

```
Genera números únicos:
ENTRANTE-2025-00001
ENTRANTE-2025-00002
SALIDA-2025-00001
```

- Nunca se repiten
- Clasificación por serie/subserie documental
- Asignación a oficinas y usuarios
- Registro completo del remitente

---

# 📧 Módulo 2: Correspondencia

## 2. Bandejas Inteligentes

**Cada usuario tiene:**

📥 **Bandeja Personal**
- Correspondencia asignada directamente a ti

📂 **Bandeja de Oficina**
- Correspondencia de toda tu área

📤 **Bandeja de Respuestas**
- Correspondencia saliente que has enviado

**Con filtros:** Estado, fecha, remitente, urgencia, etc.

---

# 📧 Módulo 2: Correspondencia

## 3. Sistema SLA (Tiempos de Respuesta)

⏱️ **Cálculo automático de fechas límite**

- Basado en calendario laboral (días hábiles)
- Clasificación: Normal, Urgente, Muy Urgente
- Alertas visuales de documentos próximos a vencer
- Integración con TRD para plazos específicos

**Ejemplo:**
- Derecho de petición → 15 días hábiles
- Solicitud urgente → 3 días hábiles

# 🏥 Casos de Uso Reales

## Escenario 1: Derecho de Petición

```
1. Ciudadano envía solicitud por email/físico
2. Ventanilla radica → ENTRANTE-2025-00150
3. Sistema asigna a Oficina Jurídica
4. Calcula fecha límite: 23/10/2025
5. Usuario de Jurídica recibe en bandeja
6. Lee y analiza solicitud
7. Crea respuesta → SALIDA-2025-00095
8. Respuesta sale en tiempo legal
9. Historial completo para auditoría
```

---

# 🏥 Casos de Uso Reales

## Escenario 2: Solicitud Interna

```
1. Facturación necesita info de Historia Clínica
2. Radica solicitud interna
3. Sistema asigna a Archivo de Historias
4. Usuario busca en FUID
5. Localiza documento (Caja 45, Carpeta 12)
6. Entrega información
7. Facturación responde
8. Trazabilidad: quién solicitó, cuándo se entregó
```

---

# 🏥 Casos de Uso Reales

## Escenario 3: Comunicación Masiva

```
1. Talento Humano envía circular a todo el personal
2. Crea comunicación masiva
3. Selecciona grupo "Todos los empleados"
4. Redacta contenido, adjunta archivos
5. Envía
6. Sistema registra: enviado a 150 destinatarios
7. Control individual de lectura/entrega
8. Estadísticas de destinatarios
```

# ❓ Preguntas Frecuentes

**P: ¿Necesitamos instalar algo en los computadores?**
R: No. Es 100% web, solo necesitas navegador y conexión.

**P: ¿Qué pasa si se cae el internet?**
R: Los datos están guardados. Al reconectar continúas donde quedaste.

**P: ¿Es difícil de usar?**
R: No. Interfaz intuitiva. Capacitación de 4 horas es suficiente.

**P: ¿Se pueden perder documentos?**
R: No. Todo queda registrado con respaldo automático.
