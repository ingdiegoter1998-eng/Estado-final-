# Normas Técnicas Colombianas (NTC) Aplicables a Correspondencia y Gestión Documental

## Panorama General

Las Normas Técnicas Colombianas (NTC) emitidas por el **ICONTEC** (Instituto Colombiano de Normas Técnicas y Certificación) establecen estándares técnicos para la elaboración, manejo y conservación de documentos. Aunque no todas son de cumplimiento obligatorio por ley, el AGN y diversas normativas las adoptan como referente técnico.

---

## NTC 3393 — Elaboración de Documentos Comerciales

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC 3393 |
| **Título** | Elaboración de Documentos Comerciales |
| **Actualización** | Quinta actualización |
| **Objeto** | Establecer principios y reglas generales para la elaboración de cartas, memorandos, circulares y demás documentos comerciales. |

### Tipos de documentos regulados:
- **Carta comercial** — Comunicación escrita que se dirige a personas naturales o jurídicas.
- **Memorando** — Comunicación interna de carácter breve.
- **Circular** — Comunicación escrita dirigida simultáneamente a varias personas.
- **Acta** — Documento que deja constancia de lo tratado y acordado.
- **Informe** — Documento que describe o da a conocer datos o hechos.

### Requisitos generales:
1. **Zonas**: Superior (membrete, datos del remitente), inferior (datos adicionales, despedida, firma).
2. **Márgenes**: Superior 3 cm, inferior 2-3 cm, izquierdo 3-4 cm, derecho 2-3 cm.
3. **Fuente**: Legible, preferentemente en tamaño 10-12.
4. **Interlineado**: Sencillo dentro de párrafos, doble entre párrafos.

## NTC 185 — Documentos Comerciales: Carta Comercial

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC 185 |
| **Objeto** | Establecer los requisitos para la elaboración de cartas comerciales. |

### Partes de una carta comercial:
1. **Código** (número de referencia o radicado).
2. **Lugar y fecha** de elaboración.
3. **Datos del destinatario** (nombre, cargo, empresa, dirección, ciudad).
4. **Asunto** (resumen breve del contenido).
5. **Saludo**.
6. **Cuerpo** del texto.
7. **Despedida**.
8. **Remitente** (nombre completo, cargo).
9. **Firma**.
10. **Anexos** (si los hay).
11. **Copia** (indicación de copias a otros destinatarios).
12. **Datos de elaboración** (iniciales del que elaboró y transcribió).

### Estilos de carta:
| Estilo | Descripción |
|--------|-------------|
| Bloque extremo | Todas las líneas parten del margen izquierdo. |
| Bloque | Fecha, despedida y firma se centran. |
| Semibloque | Primer renglón de cada párrafo con sangría. |

## NTC 3369 — Elaboración de Sobres Comerciales

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC 3369 |
| **Objeto** | Establecer los requisitos para la elaboración de sobres comerciales y su rotulación. |

### Datos del sobre:
- **Remitente**: Ángulo superior izquierdo.
- **Destinatario**: Centro del sobre.
- **Datos mínimos**: Nombre, cargo, empresa, dirección, ciudad, departamento.

## NTC 4436 — Información y Documentación: Papel para Documentos de Archivo

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC 4436 |
| **Objeto** | Establecer requisitos para el papel utilizado en documentos de archivo para conservación a largo plazo. |

### Requisitos del papel para archivo:
- **pH**: Entre 7.0 y 9.5 (papel libre de ácido/alcalino).
- **Reserva alcalina**: Mínimo 2% de carbonato de calcio.
- **Resistencia al rasgado**: Según gramaje.
- **Resistencia al doblez**: Mínimo número de dobles pliegues según gramaje.
- **Contenido de fibra**: Preferentemente fibras largas (algodón o lino para permanencia prolongada).
- **Gramaje**: Mínimo 75 g/m² para documentos de archivo.

## NTC-ISO 15489 — Gestión de Documentos

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC-ISO 15489-1 |
| **Base** | ISO 15489-1:2016 |
| **Objeto** | Establece los conceptos y principios fundamentales para la gestión de documentos. |

### Principios:
1. Los documentos son activos de la organización.
2. Se crean, capturan y gestionan como evidencia de las actividades.
3. Los sistemas de gestión documental deben ser **confiables, íntegros y conformes**.
4. Los documentos deben tener **metadatos** que describan el contexto, contenido y estructura.

### Procesos de gestión documental:
1. **Creación** — Producción de documentos como resultado de actividades.
2. **Captura** — Incorporación al sistema de gestión documental.
3. **Clasificación** — Asignación de categorías según cuadro de clasificación.
4. **Almacenamiento** — Conservación segura.
5. **Acceso** — Control de permisos.
6. **Trazabilidad** — Seguimiento de movimientos y usos.
7. **Disposición** — Transferencia, eliminación o conservación permanente.

## NTC-ISO 23081 — Metadatos para la Gestión de Documentos

| Campo | Detalle |
|-------|---------|
| **Norma** | NTC-ISO 23081-1 |
| **Base** | ISO 23081-1:2017 |
| **Objeto** | Marco para la creación, gestión y uso de metadatos en gestión documental. |

### Tipos de metadatos:
| Tipo | Ejemplos |
|------|----------|
| **De identificación** | Radicado, fecha, tipo de documento |
| **De contexto** | Remitente, destinatario, asunto, dependencia |
| **De contenido** | Descripción, palabras clave, resumen |
| **De estructura** | Formato, cantidad de folios, anexos |
| **De preservación** | Formato de archivo, suma de verificación (hash) |
| **De acceso** | Nivel de clasificación, permisos |
