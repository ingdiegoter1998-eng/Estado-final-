Guia asistida
Como enviar documentos grandes usando su Drive personal

Use esta guia cuando el sistema le indique que los adjuntos superan 25MB. El objetivo es enviar el correo sin perder el soporte documental.

Antes de continuar: quite del formulario el archivo pesado o reduzca la cantidad de adjuntos. El correo debe salir con el enlace al Drive, no con el archivo de mas de 25MB adjunto.
Cuando debe usar este proceso
  • Cuando el total de adjuntos supera 25MB.
  • Cuando necesita enviar un soporte pesado y el correo no lo permite.
  • Cuando el archivo debe quedar disponible para descarga posterior.
Resultado esperado

El destinatario recibe un correo normal y dentro del cuerpo encuentra:

  • Una explicacion breve.
  • El enlace de acceso al documento.
  • La indicacion de responder si necesita permisos.
Paso a paso detallado
1
Detenga el envio y revise los adjuntos seleccionados.
Si ya aparecio el error, no insista en enviar varias veces. Ubique el archivo pesado y recuerde que ese archivo no debe ir adjunto en el formulario final.
2
Abra su Drive personal.
Puede ser Google Drive, OneDrive u otra herramienta autorizada por la entidad. Ingrese con la cuenta que normalmente usa para compartir documentos de trabajo.
3
Suba el archivo o la carpeta.
Espere a que la carga termine por completo. No copie el enlace antes de que la plataforma confirme que el archivo ya esta disponible.
4
Configure los permisos correctamente.
Defina el acceso segun la politica interna de su dependencia. Si el destinatario debe abrir el archivo con enlace, verifique que el permiso realmente le permita entrar. Si el acceso es restringido, agregue a los destinatarios autorizados antes de copiar el enlace.
5
Copie el enlace y pruebe que funcione.
Abra el enlace en una ventana privada o en otra sesion para comprobar que no depende solo de su usuario actual. Si el enlace no abre ahi, el destinatario tampoco podra abrirlo.
6
Pegue el enlace dentro del cuerpo del correo.
Explique por que el documento va por Drive y no como adjunto. El destinatario no debe adivinar que hacer con el enlace.
7
Retire el archivo pesado del formulario y envie.
Este ultimo paso es obligatorio. Si deja el archivo pesado adjunto, el sistema volvera a rechazar el envio.
