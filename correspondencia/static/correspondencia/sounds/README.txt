ARCHIVO DE SONIDO PARA NOTIFICACIONES
=======================================

Para que las notificaciones funcionen con sonido, necesitas agregar un archivo de audio llamado:

notification.mp3

Puedes:
1. Descargar un sonido gratuito de:
   - https://mixkit.co/free-sound-effects/notification/
   - https://notificationsounds.com/
   - https://freesound.org/

2. O usar un generador de tonos online para crear un sonido simple

3. Guardar el archivo como "notification.mp3" en este directorio

Recomendaciones:
- Duración: 0.5 - 2 segundos
- Formato: MP3 o WAV
- Tamaño: Menos de 50KB
- Volumen: Moderado (el script ya tiene volumen al 50%)

Si no agregas el archivo, el sistema seguirá funcionando pero sin sonido.
